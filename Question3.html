Web Design

Web design encompasses many different skills and disciplines in the production and maintain of websites.
The different areas of web design include web graphic design, user interface design, authoring, including
standardised code and proprietary software, user experience design and search engine optimization. Often many individuals
will work on teams covering different aspects of the process, although some
designers will cover them all.
